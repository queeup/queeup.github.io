Mümkün olan her yerde `try`, `except`, `else` kullanmaya çalış.
```python
try:
    # Some Code.... 
except:
    # optional block
    # Handling of exception (if required)
else:
    # execute if no exception
finally:
    # Some code .....(always executed)
```
